## Finding and Installing new plugins

- List plugins: `bunx nx list`
- Install plugins `bunx nx add <plugin>`. Example: `bunx nx add @nx/react`.

> **Note:** This project uses **bun** as its package manager. See [coding-conventions](../../rules/development/coding-conventions.md) for import and naming rules.
